Exercice 07 - Convertisseur de degré

Créer une application qui permet de convertir une température de °C à °F et inversément.

Deux pipes, un "toFahrenheit" et "toCelsius" qui permettent de convertir une température vers une unité spécifique.

Un pipe universel qui permet de convertir vers l'unité souhaité en paramètre du pipe.

To Fahrenheit :

{{ temperature1 }} °C => {{ temperature1 | toFahrenheit }} °F

To Celsius :

{{ temperature2 }} °F => {{ temperature2 | toCelsius | number : '1.2-2' }} °C

Convertisseur Universel

Valeur convertie : {{ temperature3 | convertisseurDegre : unite | number : '1.2-2' }}
