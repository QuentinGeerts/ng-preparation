Composant enfant output

Pour définir un output dans un composant enfant, on utilise le décorateur @Output() suivi du nom de la propriété et on crée une instance de la classe EventEmitter pour émettre les événements. Le composant parent peut alors écouter cet événement en utilisant la syntaxe (nomDeLoutput)="maFonction()" dans le template où il utilise le composant enfant.
