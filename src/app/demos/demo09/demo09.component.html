Demo 08 - Les directives structurelles

Les directives structurelles sont utilisées pour manipuler la structure du DOM en fonction des données du composant

Elles sont généralement préfixées par le caractère * et permettent de créer des éléments conditionnels ou répétitifs dans le template.

Il existe plusieurs directives structurelles :

  • *ngIf : affiche un élément HTML conditionnellement en fonction d'une expression booléenne.
  • *ngFor : permet de répéter un élément HTML pour chaque élément d'un tableau.
  • *ngSwitch : permet d'afficher différents éléments HTML en fonction de la valeur d'une variable.
