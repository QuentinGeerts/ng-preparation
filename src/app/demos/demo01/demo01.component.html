Demo 01 - Property Binding (One-Way Binding)

Permet d'afficher le contenu d'une propriété directement dans le template HTML.

Affichage d'une variable entre une paire de double accolades "{{ " {{ ... }} " }}."

Unidirectionnel => TypeScript vers HTML


Valeur de maVariable1 (string) : {{ maVariable1 }}

Valeur de maVariable2 (number) : {{ maVariable2 }}

Valeur de maVariable3 (boolean) : {{ maVariable3 }}

Valeur de maVariable4 (null) : {{ maVariable4 }}

Valeur de maVariable5 (undefined) : {{ maVariable5 }}

Valeur de maVariable6 (any[]) : {{ maVariable6 }}

Valeur de maVariable7 ([string, number]) : {{ maVariable7 }}
