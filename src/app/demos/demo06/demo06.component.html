Demo 06 - Les pipes

Les pipes (ou filtres) sont des fonctions qui permettent de transformer les données affichées dans la vue. Les pipes sont utilisés pour formater, trier, filtrer ou limiter les données affichées dans la vue.

Les pipes sont utilisés dans les expressions d'interpolation ({{"{{ expression | pipe }}"}}) ou les directives de liaison de propriété ([prop]="expression | pipe").

Angular fournit un certain nombre de pipes intégrés, tels que "date", "uppercase", "lowercase", "currency", "decimal", "percent", "async", "json", etc. Il est également possible de créer des pipes personnalisés pour répondre aux besoins spécifiques de l'application.

Exemple :

String

UpperCasePipe

LowerCasePipe

TitleCasePipe

Avant : {{ maVariable1 }}

Uppercase : {{ maVariable1 | uppercase }}

Lowercase : {{ maVariable1 | lowercase }}

Titlecase : {{ maVariable1 | titlecase }}

Number

DecimalPipe

{{ "{{ value_expression | number [ : digitsInfo [ : locale ] ] }}" }}

minIntegerDigits . minFractionDigits - maxFractionDigits

Avant : {{ maVariable2 }}

Après : {{ maVariable2 | number }}

Après : {{ 123.12345 | number : '1.6-6' }}

Après : {{ 123.1 | number : '1.2-6' }}

Après : {{ 123.12345996 | number : '1.2-6' }}

Après : {{ 1234.5678 | number : '5.5-6' }}

Après : {{ 123456.789 | number : '10.2-6' : 'fr-FR' }}

Date

DatePipe

{{ "{{ value_expression | date [ : format [ : timezone [ : locale ] ] ] }}" }}

Avant : {{ maVariable3 }}

Après : {{ maVariable3 | date }}

Après : {{ maVariable3 | date : 'dd/MM/yyyy'}}

Percent

PercentPipe

{{ "{{ value_expression | percent [ : digitsInfo [ : locale ] ] }}" }}

Avant : {{ 0.2345 }}

Après : {{ 0.2345 | percent }}

Après : {{ 0.2345 | percent : '2.3-5' }}

Currency

CurrencyPipe

{{ "{{ value_expression | currency [ : currencyCode [ : display [ : digitsInfo [ : locale ] ] ] ] }}" }}

Avant : {{ 12.34 }}

Avant : {{ 12.34 | currency }}

Avant : {{ 12.34 | currency : 'EUR' }}

Avant : {{ 12.34 | currency : 'EUR' : 'symbol' }}

Avant : {{ 12.34 | currency : 'EUR' : 'code' }}

Avant : {{ 12.34 | currency : 'EUR' : 'symbol' : '4.2-2'}}

JSON

JsonPipe

{{ "{{ value_expression | json }}" }}

Avant : {{ maVariable4 }}

Avant : {{ maVariable4 | json }}
